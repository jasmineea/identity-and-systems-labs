# Rollout Plan

## Overview
This rollout plan defines the controlled steps for preparing and validating a governance-related change to the Contractors group.

---

## Rollout Steps

1. Confirm the target group and current members
2. Export current group membership using the PowerShell support script
3. Review the change objective and risk section
4. Validate the intended review/cleanup scope
5. Confirm reviewer or owner accountability
6. Prepare any membership changes for execution
7. Validate post-change state using the checklist
8. Document final outcome and notes

---

## Rollout Notes
- do not remove memberships without evidence of current state
- verify group ownership before acting
- keep change evidence with the project artifacts
