# Onboarding and Offboarding Summary

## Onboarding
The onboarding workflow in this lab included:
- creating a new user account
- assigning department-based group membership
- assigning company-wide access where appropriate
- confirming no unnecessary admin role assignment

## Offboarding
The offboarding workflow in this lab included:
- disabling the user account
- reviewing group-based access
- documenting access removal as part of the identity lifecycle

## Why It Matters
Onboarding and offboarding are foundational IAM processes. They help ensure users receive the access they need when they join and lose that access promptly when they leave.
