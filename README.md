<h1>Identity and Systems Administration Lab Portfolio</h1>

<p>
This repository contains a structured portfolio of hands-on labs focused on
Microsoft Entra, Intune, Conditional Access, identity governance, access reviews,
PowerShell, and core systems administration concepts.
</p>

<p>
The goal of this portfolio is to demonstrate practical skills across identity
and access management, governance-oriented access control, endpoint management,
documentation, scripting, validation, and security-minded operational work.
</p>

<p>
These labs were built to support roles in IAM, identity governance, Microsoft
cloud administration, endpoint administration, and junior security engineering.
</p>

<h2>Lab Areas Included</h2>

<ul>
  <li>Microsoft Entra user lifecycle management and RBAC</li>
  <li>Conditional Access and MFA policy design</li>
  <li>Enterprise application SSO and access assignment</li>
  <li>Intune endpoint compliance and configuration</li>
  <li>Identity governance, access reviews, and JML workflows</li>
  <li>Change request documentation, rollback planning, and PowerShell support</li>
</ul>

<h2>What This Portfolio Demonstrates</h2>

<p>
Across the labs in this repository, the focus is on building practical,
transferable skills in:
</p>

<ul>
  <li>least privilege access design</li>
  <li>group-based access control</li>
  <li>identity lifecycle administration</li>
  <li>access review and governance concepts</li>
  <li>change planning, validation, and rollback thinking</li>
  <li>technical documentation and audit-ready evidence</li>
</ul>

<p>
Each lab folder includes its own README, screenshots, artifacts, and notes so
that every project can stand on its own while also contributing to a broader
IAM and Microsoft administration portfolio story.
</p>
