<h1>Identity and Systems Administration Lab Portfolio</h1>

<p>
This repository contains a structured portfolio of hands-on labs focused on
Microsoft Entra, Intune, Active Directory, Conditional Access, PowerShell,
and core systems administration concepts.
</p>

<p>
The goal of this portfolio is to demonstrate practical skills across identity
and access management, endpoint management, Windows administration, scripting,
documentation, and security-minded operational work.
</p>

<p>
These labs were built to support roles in IAM, systems administration,
Microsoft cloud administration, and junior security engineering.
</p>
